Data Representation in Scatter Plot

Scatter Plot:

  • The visualization uses a scatter plot, where each data point represents a country.
  • The position of a data point along the horizontal axis encodes the daily caloric intake from carbohydrates.
  • The vertical axis encodes the daily caloric intake from fats.
  • Each point is color graded based on the average male/female BMI of that country.

Color Grading:

  • The color scale represents the average BMI of males and females in each country.
  • Users have the option to toggle between male and female BMI views.
  • Countries with a female BMI average are colored in shades of red, while male BMI averages are in shades of blue.

Interactions:

  • Hover Tooltip The tooltip provides additional information:
  • Upon hovering over a data point, the tooltip displays the following:
    • Country Name
    • X Value: Daily Caloric Intake from Fat
    • Y Value: Daily Caloric Intake from Carbohydrates
    • BMI Value of the selected gender (male or female)
